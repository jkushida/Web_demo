このリポジトリは、さまざまなウェブデモをまとめたものです。

デモ一覧

データ構造とアルゴリズム

再帰・状態空間探索

  • Water Jug Problem - 4Lと3Lの水差しで2Lを作るシミュレーショ ン
  • Stairs Recursion - 再帰による階段の登り方問題のデモ
  • Tower of Hanoi - ハノイの塔のインタラクティブなデモ
  • River Crossing - 川渡り問題のシミュレーション

探索・数え上げ・ハッシュ

  • Euclidean Algorithm Visualizer - ユークリッドの互除法で最大公約数を求める過程を可視化
  • Sentinel Search Comparison - 番兵あり・なしの線形探索を並べて、要素比較と範囲確認の差を可視化
  • Hash Search - ハッシュ法の探索例と衝突処理を、オープンアドレス法/チェイン法で可視化
  • Maze Search (BFS/DFS) - 幅優先/深さ優先で迷路を探索。探索木とログを動的表示

線形データ構造

  • Singly Linked List - 単方向リストの挿入・削除で、next ポインタのつなぎ替えをステップごとに可視化

木・グラフ

  • Huffman Coding - 出現頻度からハフマン木を構築し、符号表・平均 符号長・エントロピーHを可視化
  • Graph Visualizer - BFS/最短経路などの可視化
  • BST / AVL Rotation Visualizer - 同じ挿入列で BST と AVL を並べ、回転と木の高さの差を可視化

ソート

  • Bubble Sort Animation - 配列のスワップ移動と疑似コードの対応でバブルソートを可視化
  • Selection Sort Animation - 未ソート部分から最小値を探し，先頭と交換する選択ソート
  • Insertion Sort Animation (ins/cmp/temp Ver.) - 後ろへまとめてずらして挿入する版
  • Insertion Sort Animation (Swap Ver.) - 隣接交換で1データずつ前へ送る挿入ソート
  • Heap Sort Visualizer - 最大ヒープの構築からソートまでの過程を木構造で可視化
  • Bucket Sort Animation - 比較によらず、値を対応する bucket に振り分けてから順に回収する手順を可視化
  • Radix Sort Animation - 比較によらず、1の位から100の位まで前の桁の順序を保ちながら並べ替える
  • Count Sort Animation - 比較によらず、counts と累積和を使って tmp を作る流れを可視化

進化計算

  • GA OneMax - OneMax問題を解く遺伝的アルゴリズムのデモ
  • CMA-ES Simulator - CMA-ESのシミュレータ
  • DE Simulator - 差分進化のシミュレータ
  • PSO Simulator - 粒子群最適化のシミュレータ
