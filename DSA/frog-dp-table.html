DPテーブル

dp[i] は足場 i へ行く最小コスト
dp[i] = min(dp[i-1] + |h[i]-h[i-1]|, dp[i-2] + |h[i]-h[i-2]|)

擬似コード

dp[0] = 0
for i = 1 ... n-1:
cost1 = dp[i-1] + abs(h[i] - h[i-1])
if i >= 2:
cost2 = dp[i-2] + abs(h[i] - h[i-2])
dp[i] = min(cost1, cost2)
else: dp[i] = cost1
