クイックソートのコード

中央の要素を pivot とし、分割後の左側と右側を再帰的に同じように処理します。
  1. 1def quick_sort(a, left, right):
  2. 2  pivot = a[(left + right) // 2]
  3. 3  pl, pr = partition(a, left, right, pivot)
  4. 4  if left < pr:
        quick_sort(a, left, pr)
  5. 5  if pl < right:
        quick_sort(a, pl, right)
  6. 6  return a
