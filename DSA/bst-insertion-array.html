二分探索木への追加（配列表現）

手順 0 / 0
親より小さい値は左、大きい値は右。同じ値は擬似コードの else に合わせて右。

木の接続状態

rootIdx = 0

配列 tree[10]

data / left / right

挿入列をセット。addBST(data) の処理を1行ずつ追跡。

擬似言語 addBST(data) と main()

行ハイライト
準備: 構造体とグローバル変数 # ノードの構造 BST { data, left, right } tree[10] # 二分探索木を格納する配列 rootIdx ← 0 # 根の添字 newIdx ← 0 # 次に使う空き添字
1関数 addBST(data) # 1つ追加 2 tree[newIdx].data ← data # 値を格納 3 tree[newIdx].left ← -1 # 左は空 4 tree[newIdx].right ← -1 # 右も空 5 if newIdx != rootIdx # 根でなければ探索 6 currentIdx ← rootIdx # 根から開始 7 addFlag ← false # まだ未追加 8 while addFlag == false # 接続まで反復 9 if data < tree[currentIdx].data # 小さいなら左 10 if tree[currentIdx].left == -1 # 左が空? 11 tree[currentIdx].left ← newIdx # 左に接続 12 addFlag ← true # 追加完了 13 else # 左に子あり 14 currentIdx ← tree[currentIdx].left # 左へ移動 15 else # 大きいなら右 16 if tree[currentIdx].right == -1 # 右が空? 17 tree[currentIdx].right ← newIdx # 右に接続 18 addFlag ← true # 追加完了 19 else # 右に子あり 20 currentIdx ← tree[currentIdx].right # 右へ移動 21 newIdx ← newIdx + 1 # 次の空きへ
