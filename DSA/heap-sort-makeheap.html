Heap Sort Visualizer（makeHeap / sift-up）

  1. 先頭から末尾へ1つずつ heap に追加 する
  2. 追加した子を親と比べ、必要なら 上へ交換 する
  3. 根（最大値）と末尾を 交換 する
  4. 残ったヒープを downHeap で再構築する
makeHeap は子→親（下から上へ）

現在の配列

makeHeap: data[0] から順に heap へ追加し、追加した子を親方向へ上げます。

  1. 1○ 副プログラム: makeHeap( 整数型: data[], 整数型: heap[], 整数型: hnum )
  2. 2○ 整数型: i, k
  3. 3■ i: 0, i < hnum, 1
  4. 4・heap[i] ← data[i] /* heap にデータを追加 */
  5. 5・k ← i
  6. 6■ k > 0
  7. 7▲ heap[k] > heap[parent(k)]
  8. 8・swap(heap, k, parent(k))
  9. 9・k ← parent(k)
  10. 10・break /* 内側の繰返し処理から抜ける */
  11. 1○ 副プログラム: heapSort( 整数型: data[], 整数型: heap[], 整数型: hnum )
  12. 2○ 整数型: last
  13. 3・makeHeap(data, heap, hnum)
  14. 4■ last: hnum - 1, last > 0, -1
  15. 5・swap(heap, 0, last) /* heap[0] と heap[last] の値を交換 */
  16. 6・downHeap(heap, last - 1) /* heap を再構成 */
  17. 1○ 副プログラム: downHeap( 整数型: heap[], 整数型: hlast )
  18. 2○ 整数型: n, tmp
  19. 3・n ← 0
  20. 4■ lchild(n) ≦ hlast
  21. 5・tmp ← lchild(n)
  22. 6▲ rchild(n) ≦ hlast
  23. 7▲ heap[tmp] ≦ heap[rchild(n)]
  24. 8・tmp ← rchild(n)
  25. 11▲ heap[tmp] > heap[n]
  26. 12・swap(heap, n, tmp)
  27. 13・return /* downHeap から抜ける */
  28. 15・n ← tmp
