パスカルの三角形と再帰関数

組合せ nCr の再帰式を使い、パスカルの三角形の各値がどのような再帰呼び出しで求まるかを可視化します。 三角形の数をクリックすると、追跡する c(n, r) を変更できます。

0 / 0

両端の値

nC0 = nCn = 1

内側の値

nCr = n-1Cr-1 + n-1Cr

二項定理

(a + b)n の係数は nCr

パスカルの三角形

色が再帰ツリーの状態と対応します。

選択した値

5C2 = 10

Pythonコード

  1. def c(n, r): # nCr を求める関数
  2. if r == 0 or n == r: # 三角形の両端か調べる
  3. return 1 # 両端の値は 1
  4. return c(n - 1, r - 1) + c(n - 1, r) # 左上と右上を足す
  5.  
  6. # main
  7. n = int(input()) # n を入力
  8. r = int(input()) # r を入力
  9. value = c(n, r) # main から関数を呼び出す
  10. print(value) # 求めた値を表示
