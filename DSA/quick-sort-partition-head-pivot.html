クイックソート - 先頭要素を基準値にする分割

教科書の分割（基準値 = a[head]）:
  • 先頭要素 a[head] を基準値とし、left = head + 1、right = tail から調べる
  • 左から基準値より大きい要素、右から基準値以下の要素を探す
  • left < right の間は、見つけた2要素を交換して探索を続ける
  • 最後に a[head] と a[right] を交換し、基準値の位置 right を返す
— / —

任意の配列: 「配列 a」の入力欄をカンマ区切りで書き換え、「リセット」を押します。 例: 1, 2, 3, 4, 5, 1

教科書の分割コード

  1. 1def divide_array(a, head, tail): # 分割関数
  2. 2 left = head + 1 # 左位置
  3. 3 right = tail # 右位置
  4. 4 pivot = a[head] # 基準値
  5. 5 while True: # 探索を繰り返す
  6. 6 while left <= tail and pivot >= a[left]: left += 1 # 左探索
  7. 7 while pivot < a[right]: right -= 1 # 右探索
  8. 8 if left >= right: break # 交差で終了
  9. 9 a[left], a[right] = a[right], a[left] # 交換
  10. 10 left += 1; right -= 1 # 内側へ移動
  11. 11 a[head], a[right] = a[right], a[head] # 基準値を確定
  12. 12 return right # 確定位置を返す
