クイックソート - 分割の決め方

分割（パーティション）の決め方:
  • 配列 a から枢軸（pivot）を選択する（ここでは中央の要素）
  • a[pl] ≥ pivot となる値を左から探す
  • a[pr] ≤ pivot となる値を右から探す
  • pl と pr が交差するまで交換を繰り返し、pivot 以下／以上のグループに分ける
